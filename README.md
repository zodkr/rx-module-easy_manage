## 운영도우미 모듈 (Rhymix 2.1)

다음과 같은 기능들을 지원합니다

- Slack 채널에 웹훅 전송
  - 게시물/댓글 신고가 들어왔을때
  - 특정 게시판에 게시물이 작성됐을때
- 사용자에 대한 관리 메모 바로 추가
- 관리자 액션에 대한 로그 추가
- 회원에 대한 검증액션 추가

해당 모듈은 zod.kr의 운영상 필요한 기능들만 추가되었으며, 풀리퀘스트와 같은 직접적인 기능 추가 요청이 아닌 이상 별도의 기능추가 요청은 받고있지 않습니다.
